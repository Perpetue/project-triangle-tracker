Project Triangle Tracker

This project is about making the interface logic and business logic where the interface logic is a form made by three input places where the end user have to enter three different numbers which are the three lengths of the triangle and one button where the end user click to find which type of triangle.

Equilateral Triangle:There has three sides are equal

Isoceles triangle:There has three sides but two sides are equal.

Scalene triangle:There has three sides but all sides aren't equal.

Instruction of the User

If Lenght1= Lenght2 and Lenght1= Lenght3 and Lenght2= Lenght3,the user has an equilateral triangle.

If Lenght1= Lenght2 or Lenght1= Lenght3 or Lenght2= Lenght3, the user has an isoceles triangle.

If all the above two conditions are not now it is a scalene triangle.
